
# Fabric

## Using Custom Patterns
```
# Setup 
# fabric -S, --setup
fabric -S

# Choose [11]
# Provide repository, e.g. https://github.com/tmsdnl/fabric.git
# Provide folder name if different from patterns

# List new patterns
# fabric -n, --latest=
fabric -n 10

# OR check
ls ~/.config/fabric/patterns

# Update patterns from configured repository
# fabric -U, --updatepatterns
fabric -U
```
